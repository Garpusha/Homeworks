1. Программа берет настройки доступа к БД и путь к файлам CSV для первоначальной загрузки из файла config.ini
2. При запуске старая база обнуляется
3. Программа не запрашивает данные пользователей при внесении новых записей и модификации существующих, а создает из шаблонов с помощью генератора случайных чисел
4. Количество телефонов у одного пользователя определяется случайно в диапазоне 0-2
5. При изменении данных пользователя будет создано новое имя и фамилия, а также e-mail. При при изменении телефона будет содано столько же номеров телефонов, сколько было изначально
